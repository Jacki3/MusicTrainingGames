Synth Strings #2
----------------

Version 2020-05-28

General MIDI instrument #52: Synth Strings 2. Made by roberto@zenvoid.org for
the FreePats project, with ZynAddSubFX / Yoshimi software synthesizers.
ZynAddSubFX and Yoshimi are free software programs capable of creating many
different sounds with several synthesis engines, filters and effects.

This sound bank, in several formats and qualities, is available on the
FreePats project web pages: http://freepats.zenvoid.org/
