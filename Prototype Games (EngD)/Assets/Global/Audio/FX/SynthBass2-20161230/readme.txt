Release 2016-12-30 (version 1).

Sound samples and SFZ/SF2 sound banks made by roberto@zenvoid.org for the
FreePats project. There are other versions of this soundfont in the FreePats
web site: http://freepats.zenvoid.org/

Imitation of the Yamaha DX7 bass sound, a FM sythesis instrument that was
popular in the 1980s. These sound samples simulate the "BASS 1" patch of the
DX7, used in many 1980s popular songs.

Recorded from Hexter software synthesizer. Hexter is a free software program
that accurately models the sound generation algorithm of the original
instrument: http://dssi.sourceforge.net/hexter.html
